# HTTP-Ray-Trace
## How To Use:
1. Compile with "make"       
2. Run server with ./main
3. Go to [Raytracer](https://web.eecs.utk.edu/~smarz1/courses/cosc360/rtserver.html) and enter server ip (put 8177 for default)
4. Enter scene information and click "send to server" to render image in your specified format.
